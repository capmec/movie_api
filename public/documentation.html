Documentation page

Objective

To build the server-side component of a “movies” web application.
The web application will provide users with access to information about different movies, directors, and genres.
Users will be able to sign up, update their personal information, and create a list of their favorite movies.

API Functions Description URL Endpoints Descriptions
Get All Movies GET /movies Return a list of ALL movies to the user
Get One Movie GET /movies/movie%20title Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
Get a Genre GET /movies/genre/Genre Return data about all movies titles with a specific genre
Get a Director GET /directors/Director%20Name Return data about a director (bio, birth year, death year) by name
Create a New User POST /users Allow new users to register
Update User Info PUT /users/:id Allow users to update their user info (username)
Add Movie to User's Favorites PUT /users/:id/favoriteMovie Allow users to add a movie to their list of favorites (showing only a text that a movie has been added)
Remove a Movie from User's Favorites DELETE /users/:id/favoriteMovie Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed)
Delete User DELETE /users/:id Allow existing users to deregister (showing only a text that a user email has been removed)
